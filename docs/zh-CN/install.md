# LinkClawCN Android 安装指南

本文档介绍如何在 Android 设备上安装和使用 LinkClawCN 移动客户端。

## 系统要求

- **Android 版本**: Android 12 (API 31) 或更高版本
- **存储空间**: 约 100 MB
- **网络**: 需要网络连接以连接到 Gateway

## 安装方式

### 方式一：从 GitHub Release 下载 APK

1. 访问 [LinkClawCN Releases](https://github.com/gzxh-ll/linkclawcn/releases)
2. 下载最新的 `LinkClawCN-android-*.apk` 文件
3. 在手机上打开 APK 文件
4. 如果系统提示"安装被阻止"，请启用"允许安装来自未知来源的应用"
5. 按照屏幕提示完成安装

### 方式二：从应用商店安装（待上线）

## 权限说明

LinkClawCN Android 应用需要以下权限：

| 权限 | 用途 | 必需 |
|------|------|------|
| 网络访问 | 连接到 Gateway | 是 |
| 相机 | 扫描二维码、拍照 | 否 |
| 麦克风 | 语音对话 | 否 |
| 位置 | 发现附近的 Gateway | 否 |
| 通知 | 接收消息通知 | 否 |
| 存储 | 保存和读取文件 | 否 |

### 权限申请说明

首次启动时，应用会根据功能需求请求相应权限：

- **相机权限**: 用于扫描 Setup 二维码或使用相机相关功能
- **麦克风权限**: 用于语音对话功能
- **位置权限**: 用于发现局域网中的 Gateway 设备
- **通知权限**: 用于接收新消息推送

您可以在设置的"权限"页面随时管理这些权限。

## 连接方式

### 方式一：Setup 码连接（推荐）

1. 在电脑上启动 Gateway：`openclaw gateway --port 18789`
2. 打开手机 App，进入"连接"标签
3. 选择"Setup 码"模式
4. 在电脑上运行 `openclaw nodes pending` 查看配对请求
5. 在电脑上运行 `openclaw nodes approve <请求ID>` 批准配对

### 方式二：手动连接

1. 在电脑上启动 Gateway：`openclaw gateway --port 18789`
2. 打开手机 App，进入"连接"标签
3. 选择"手动输入"模式
4. 输入以下信息：
   - **主机地址**: Gateway 所在设备的 IP 地址（如 `192.168.1.100`）
   - **端口**: `18789`（默认）
   - **TLS**: 关闭（默认）
5. 点击"连接"按钮

### 方式三：USB 直连（开发测试）

1. 确保手机通过 USB 连接到电脑
2. 在电脑上启用 USB 调试
3. 运行 `adb reverse tcp:18789 tcp:18789`
4. 在手机 App 中输入：
   - **主机地址**: `127.0.0.1`
   - **端口**: `18789`

## 常见问题

### 连接失败

**问题**: 无法连接到 Gateway

**解决方案**:
1. 确保手机和 Gateway 在同一网络下
2. 检查 Gateway 是否正在运行：`pnpm openclaw gateway --port 18789`
3. 检查防火墙是否阻止了 18789 端口
4. 尝试使用手动输入模式，直接输入 Gateway 的 IP 地址

### 配对失败

**问题**: 配对请求被拒绝或超时

**解决方案**:
1. 确保在电脑上运行了 `openclaw nodes approve` 命令
2. 检查电脑上的配对请求：`openclaw nodes pending`
3. 尝试重新发起配对

### 消息无法发送

**问题**: 发送消息失败或无响应

**解决方案**:
1. 检查网络连接状态
2. 确保已成功连接到 Gateway
3. 查看手机日志获取详细错误信息

### 应用闪退

**问题**: 打开应用后立即闪退

**解决方案**:
1. 确保 Android 版本不低于 12 (API 31)
2. 尝试卸载后重新安装
3. 清除应用缓存：设置 → 应用 → LinkClawCN → 清除缓存

### 语音功能无法使用

**问题**: 无法使用语音对话

**解决方案**:
1. 确保已授予麦克风权限
2. 检查手机音量设置
3. 确保 Gateway 支持语音功能

## 功能说明

### 主要功能

- **连接管理**: 连接和断开 Gateway
- **聊天**: 发送和接收消息
- **语音**: 语音对话（需麦克风权限）
- **屏幕共享**: 查看和控制屏幕（实验性）

### 设置选项

在"设置"标签中可以配置：

- Gateway 连接设置
- 通知偏好
- 语音设置
- 外观主题（浅色/深色/跟随系统）
- 语言设置

## 卸载

1. 打开手机"设置"
2. 进入"应用"或"应用管理"
3. 找到"LinkClawCN 节点"
4. 点击"卸载"

卸载后，所有本地数据将被清除。
